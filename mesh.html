📡 Связь на поляне без интернета

На поляне сети нет, но есть mesh: приложение Bitchat гоняет текстовые сообщения по Bluetooth от телефона к телефону. Чем больше людей его поставит, тем дальше бьёт сеть — сообщения прыгают через чужие телефоны.

1 Установи ДО выезда — на поляне будет поздно

Android — прямая ссылка на APK:

⬇ Скачать APK (Bitchat 1.7.4) браузер спросит разрешение на установку — это нормально для приложений не из маркета · работает только с интернетом

iPhone — App Store, поиск «Bitchat Mesh»:

App Store — Bitchat Mesh работает только с интернетом

2 Открой приложение и разреши Bluetooth

Больше ничего не нужно: ни регистрации, ни номера телефона. Ник можно сменить тапом по нему.

3 Проверь две настройки

  • канал #mesh — выбран по умолчанию, это общий эфир поляны;
  • тумблер run in background — ВКЛЮЧЁН. Без него связь умирает, как только приложение свернулось.

4 Что НЕ трогать

Geohash-каналы, карту локаций и Tor — это городские фичи, они работают через интернет и на поляне бесполезны. Доступ к локации приложению давать не нужно.

💬 Как этим пользоваться на фесте

  • #mesh — общий эфир: видно всех в радиусе, сообщения прыгают дальше через телефоны других пользователей;
  • личка — тап по нику в списке Сеть → Люди;
  • для своей компании договоритесь о префиксе в квадратных скобках, например [ЛО] выезжаем к сцене — отдельных каналов в mesh-режиме нет, префикс легко ищется глазами.

📍 Обмен метками

Кинули вам координаты лагеря текстом в Bitchat? Скопируйте сообщение и вставьте в форму — приложение само найдёт координаты и название:

➕ Добавить метку из текста

а своей меткой можно поделиться из её карточки на карте — ссылка вида #pin=… открывается даже без сети у того, у кого приложение уже стоит

Честно: Bluetooth-mesh — это 10–50 метров на прыжок, в чистом поле дальше, в толпе меньше. Сообщения могут идти долго или не дойти. Bitchat — стороннее приложение, мы его не делали: вопросы не к нам 😄 Но на прошлых фестах штука работала.
